# solana-gas-tracker
Solana gas tracker lib written in Golang

## Concept

1. Connect to a node on QuickNode.
2. Call the "compute budget" method.
3. Calc gas fee.

They also have an API, but it's paid:
https://www.quicknode.com/gas-tracker/solana
